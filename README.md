# SchoolProjects

__At initial download from Git, run commands:__
> python -m venv venv \
> .\venv\Scripts\activate \
> pip install flask \
> pip install watchdog \
> npm install sass \
> python static\python\sass_watcher.py --full-sass \
> _CTRL + C_ to terminate


__To activate server:__
> python app.py \
> _CTRL + C_ to terminate
